# mug create

Creates the boilerplate for your AWS Lambda for golang project.

### Synopsis

Creates the boilerplate for your AWS Lambda for golang project.

```
mug create [flags]
```

### Options

```
  -h, --help            help for create
  -r, --region string   Region the project will be deployed to
                        (e.g. us-east-1 or eu-central-1) (default "eu-central-1")
```
