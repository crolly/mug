    # mug

    microservices understand golang - easily creating serverless AWS Lambda CRUDL apps

    # Synopsis

    mug lets you create AWS Lambda for golang projects and boilerplates the project structure with serverless configuration and a required Makefile to build the functions. You can easily add CRUDL functions as resources.
